# Custom Prompts

This directory contains Kiro CLI prompts for DeploySim development.

## Project-Specific Prompts

| Prompt | Purpose |
|--------|---------|
| `DEVLOG.md` | Session tracking with timestamps, duration, and Kiro credit usage |
| `code-review-hackathon.md` | Review against official Kiro Hackathon judging criteria |
| `dispatcher-debug.md` | Debug VibeSprint issues |
| `dispatcher-extend.md` | Extend VibeSprint with new executor |

## Development Workflow Prompts

| Prompt | Purpose |
|--------|---------|
| `plan-feature.md` | Create comprehensive specs with agent assignments for parallel execution |
| `execute.md` | Execute implementation plans task-by-task |
| `prime.md` | Load project context before starting work |
| `daily-next-step.md` | Identify next priority task from roadmap |

## Quality & Review Prompts

| Prompt | Purpose |
|--------|---------|
| `code-review.md` | Technical code review for bugs and quality |
| `code-review-fix.md` | Fix issues found in code reviews |
| `system-review.md` | Analyze implementation vs plan for process improvements |
| `security-checklist.md` | Security review checklist |

## Issue Management Prompts
